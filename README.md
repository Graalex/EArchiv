EArchiv
=======
Проект электронного архива, для компании-оператора газораспределительных систем.
Построен по модульному принципу, с использование фреймворка Microsoft Prism.

Структура проекта.
_________________
### 1.AdmTools #
Программа с графическим пользовательским интерфейсом для выполнения административных
задач.

### 2.EArchiv #
Программа с графическим пользовательским интерфейсом для работы с электронными копиями документов

### 3.EContract #
Программа с графическим пользовательским интерфейсом для работы с электронными копиями договоров.

### 4.Mariupolgaz.EArchiv.Common #
Библиотека содержащая реализацию предметной модели данных и интерфейсы сервисов, используемые в 
электронном архиве

5.Mariupolgaz.EArchiv.Abonent
6.Mariupolgaz.EArchiv.DocsAbon
7.Mariupolgaz.EArchiv.Document
8.Mariupolgaz.EArchiv.Finder
9.Mariupolgaz.EArchiv.Security
